# Home2Dream Explorer

3D globe viz for hometown → dream home journeys (Rocket · Super Bowl LX Fan Experience).

## Run locally

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Push to a GitHub repo.
2. **Settings → Pages → Build and deployment**: choose **GitHub Actions**.
3. Push to `main` to trigger a deploy.

The app will be live at `https://<username>.github.io/<repo-name>/`.
